# Fonts

Place **`KFGQPC_Uthmanic_Hafs.ttf`** here (the primary Arabic face, PRD 4.1),
then uncomment the `fonts:` block in `pubspec.yaml`.

Source: KFGQPC fonts are published via the King Fahd Quran Printing Complex and
mirrored on QUL (https://qul.tarteel.ai/resources/quran-script). Confirm the
font licence before shipping (see the licensing note in the alquran-data repo).

`Kitab` is the planned alternate naskh face.
